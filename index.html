SVG

Scalable vector graphics

SVG Advantages

  • Resolution Independent
  • Reducing HTTP Request
  • Can be animated and Edited
  • Smaller File Size

How to start?

  1. Insert svg tay into your html document
  2. Specify the size of your svg with without height
  3. Insert some shapes into your svg

Basic SVG Shapes

  • Circle
  • Rectangle
  • Ellipse
  • Line
  • Polygon
  • Polyline
